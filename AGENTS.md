# Scrollect

## What it is

Scrollect is an AI-powered personal learning feed. It transforms saved content (books, articles, videos, PDFs) into a scrollable feed of bite-sized learning cards — like social media, but built from your own knowledge.

## Target user

Software engineers and active learners who consume a lot of content but struggle to retain and connect what they learn.

## Key principles

- **Personal, not social** — no followers, no comments, no public profiles
- **Scroll-native UX** — don't fight the scroll, redirect it
- **AI-first** — the feed is generated by a dedicated AI agent
- **Low friction input** — adding content should take seconds

## Tech stack

- **Monorepo**: Turborepo + Bun
- **Web app**: Next.js (`apps/web`)
- **Docs**: Starlight (`apps/docs`)
- **Presentation**: Slidev (`apps/presentation`)
- **Backend**: Convex (`packages/backend`)
- **Auth**: Better-Auth
